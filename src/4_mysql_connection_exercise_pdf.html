Exercise: Setting up the MySQL Connection

Step 1

Install and Start MySQL Server

To install MySQL server, download the installer appropriate for your operating system from https://www.mysql.com/downloads/.

Start the MySQL server. It runs on port number 3306 by default.

To confirm it is running, start the MySQL command line client with the following command:

mysql -u root -p
Note: You will be prompted to enter the root password you set during MySQL installation. If you didn't set a password, just press Enter.
Step 2

Install MySQL DB API Driver

Activate the Python virtual environment you are using for this project, and open VS Code in it. Follow the instructions from the earlier exercise.

Install the MySQL DB API Driver, mysqlclient from the CMD terminal in VS Code, use PIP utility with following command:

pip3 install mysqlclient
Important: On Windows, you may need to install Microsoft Visual C++ Build Tools if you encounter installation errors. Alternatively, you can use pip3 install pymysql as an alternative driver.
Step 3

Configure Database Settings

Open the settings.py file in a VS Code window. Locate the DATABASES section. In place of the existing SQLite configuration, paste the following configuration for MySQL.

settings.py
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'LittleLemon',
        'USER': 'root',
        'PASSWORD': '',
        'HOST': '127.0.0.1',
        'PORT': '3306',
        'OPTIONS': {
            'init_command': "SET sql_mode='STRICT_TRANS_TABLES'"
        }
    }
}
Note: Make sure to replace the empty PASSWORD string with your MySQL root password if you set one during installation.

Before running migrations, you need to create the database. Connect to MySQL and run:

mysql -u root -p
CREATE DATABASE LittleLemon;
EXIT;
Step 4

Install MySQL Extension

Install MySQL extension from VS Code extension marketplace.

  1. Open VS Code Extensions panel (Ctrl+Shift+X on Windows/Linux or Cmd+Shift+X on Mac)
  2. Search for "MySQL" by Jun Han
  3. Click Install
Step 5

Run Database Migrations

From the CMD terminal on Windows (Command palette in Mac OS) inside VS Code, run the migrate command.

python manage.py migrate

The table structure required for the INSTALLED_APPS will be created in the MySQL database named LittleLemon.

Expected Output: You should see messages indicating that migrations are being applied, such as:
  • Applying contenttypes.0001_initial... OK
  • Applying auth.0001_initial... OK
  • Applying admin.0001_initial... OK
  • And more...
Step 6

Connect to Database in VS Code

Note that the MySQL tab appears in the file explorer bar of VS Code.

  1. Click on the + button and connect to the database
  2. Enter localhost as the domain name
  3. Enter your MySQL username (typically root)
  4. Enter your password (or leave blank if you didn't set one)

Now localhost appears in the MySQL tab.
